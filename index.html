Bienvenidos a la presentación de Tipos de archivos

Bienvenidos a la presentación de Tipos de archivos

Vamos a revisar la definición de archivos planos, archivos xml, archivos json y por último archivos parquet.

Archivos planos


Archivos planos

Los archivos planos son archivos de texto sin formato, que contienen información en una secuencia de caracteres y líneas. Estos archivos son fáciles de leer y editar por los humanos, pero no son eficientes para el procesamiento de grandes cantidades de datos o para compartir información estructurada.

Archivos XML


Archivos XML

Los archivos XML son archivos de texto estructurados que utilizan etiquetas para definir elementos y atributos para definir sus propiedades. Estos archivos se utilizan para intercambiar información entre diferentes sistemas y aplicaciones, y se pueden utilizar para describir datos complejos y jerárquicos.

Archivos JSON


Archivos JSON

Los archivos JSON (JavaScript Object Notation) son archivos de intercambio de datos que se utilizan para almacenar y transmitir información estructurada entre diferentes sistemas y aplicaciones. Estos archivos utilizan pares de clave-valor para representar la información, y son muy populares en la industria de la tecnología de la información.

Archivos Parquet


Archivos Parquet

Los archivos Parquet son archivos diseñados para mejorar el rendimiento de las consultas en sistemas de procesamiento de grandes datos. Estos archivos almacenan los datos en columnas en lugar de filas, lo que permite una compresión más eficiente y una mayor velocidad de lectura. Se utilizan ampliamente en la industria para almacenar y analizar grandes cantidades de datos.
